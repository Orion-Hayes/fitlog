# FitLog

一个轻量、无登录的健身训练记录工具。它可以按动作分类记录组数、次数、重量、时长和备注，并在本地保留训练历史与统计数据。

## 主要功能

- 快速记录一组训练
- 按胸部、背部、腿部等分类管理动作
- 添加自己的常用动作
- 查看训练历史和累计统计
- 导出、导入 JSON 数据备份
- 支持安装到手机主屏幕

## 使用方式

直接打开 `index.html` 即可使用。训练数据只保存在当前浏览器中；换手机或清理浏览器前，请先通过右上角的导出按钮备份数据。

## 发布到 GitHub Pages

将本项目上传至 GitHub 后，在仓库的 `Settings` → `Pages` 中选择从 `main` 分支的根目录发布。发布完成后会得到一个 HTTPS 地址；用手机打开它即可安装到主屏幕。

## 项目文件

- `index.html`：页面结构
- `styles.css`：界面与手机端布局
- `app.js`：训练记录、动作库与数据逻辑
- `manifest.webmanifest`、`sw.js`：手机安装和离线支持
